Now let's see two new type of elements, the textarea and select tag.

The textarea is used when you want to allow the user to write a large amount of text like an article or a review. The height attribute specifies the visible height of the text.
This is a text area where you can write anything that you want. The number of row and columns is defined by the rows and cols attributes. This is used to make a post for example. It is maintly use for taping text.
By specifies the option resize to none, it is possible to disable resizing with mouse or touch events(style=" resize: none").

The select tag us used to select an element between a list of element or list of element grouped by category. The attribute "name" is required for this kind of element.
